A community-driven collection of **tools, scripts, and dashboards** for Microsoft Intune & Endpoint Manager.  
This list brings together trusted **open-source and community tools** used by Intune administrators to **diagnose, automate, monitor, secure, and document** enterprise environments.

---

## 🎯 Purpose

Managing Intune at scale requires more than the built-in admin portal.  
These tools help administrators to:

- Troubleshoot enrollment, sync, and deployment failures  
- Analyze Intune, IME, and Autopilot logs efficiently  
- Automate repetitive operational tasks  
- Audit assignments, RBAC roles, and configuration changes  
- Clean up stale or orphaned devices  
- Generate documentation for audits and compliance  

---

## 🧰 Tools

### 🔧 Intune Debug Toolkit
Classic troubleshooting kit with live log viewers, re-run of failed processes, and helpers to diagnose deployment and sync issues.  
👉 https://github.com/MSEndpointMgr/IntuneDebugToolkit

---

### ⚡ Autopilot Diagnostics (Community)
PowerShell script showing Windows Autopilot setup events step by step. Helps identify where device enrollment failed.  
👉 https://www.powershellgallery.com/packages/Get-AutopilotDiagnosticsCommunity

---

### 📦 IME Diagnostics
Detects issues with the Intune Management Extension (IME) during app and script deployments.  
👉 https://github.com/petripaavola/Get-IntuneManagementExtensionDiagnostics

---

### 🛠 IntuneDiag
Comprehensive Microsoft toolkit for troubleshooting Intune & Endpoint Manager (logs, compliance, apps, sync).  
👉 https://github.com/microsoft/IntuneDiag

---

### 📋 IntuneManagement
GUI tool for policy migration and tenant setup. Supports import/export of Intune configurations.  
👉 https://github.com/mickekarlsson/IntuneManagement

---

### 🧹 Device Offboarding Manager
Find and bulk-delete stale or orphaned devices from Intune, Autopilot, and Entra ID.  
👉 https://github.com/ugurkocde/DeviceOffboardingManager

---

### 🤖 Intune Automation
Collection of PowerShell automation scripts for lifecycle management, compliance, and reporting tasks.  
👉 https://github.com/ugurkocde/IntuneAutomation

---

### 🔐 TenuVault
Secure vault for storing Intune secrets, credentials, and sensitive configurations.  
👉 https://github.com/ugurkocde/tenuvault

---

### 📚 Intune Scripts Collection
Large library of ready-to-use PowerShell scripts for Intune device management and automation.  
👉 https://github.com/JayRHa/Intune-Scripts

---

### 📊 Intune Assignment Checker
Generates HTML reports with charts showing app and policy assignments across users, groups, and devices.  
👉 https://github.com/ugurkocde/IntuneAssignmentChecker

### 📝 Intune Change Tracking

### 📈 Intune Monitoring Dashboards
Prebuilt Azure Monitor Workbooks for compliance, device health, and app deployments.

### 🛡 Intune RBAC Auditor
